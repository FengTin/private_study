1、把项目推送到暂存区
    git add "文件名称"
    git add *   // 添加全部文件
2、 把项目推送到历史区
    git commit -m"注释"

3、 在本地回退或者切换版本
    git reset --hard "版本号"

4、 把项目推送到远程仓库
    git push "远程仓库的地址" master

其他命令：
1、注册
    git config --global user.name 用户名
    git config --global user.email 用户油箱
2、查看状态
    git status
3、查看历史区
   git log
4、删除
   git rm "文件名"
5、把远程仓库的代码拉取到本地
    git pull "远程仓库的地址" master  --allow-unrelated-histories
6、如果pull不下来，在后面添加 --allow-unrelated-histories

分支
1、在本地创建分支，切换分支dev
    git branch dev

2、切换分支
    git checkout dev

3、创建以及切换分支
    git checkout -b dev

4、在远程仓库创建分支dev，并且从本地分支dev向远程仓库分支dev提交代码

5、在远程仓库把分支dev代码合并到主分支master

6、远程仓库 码云

7、ssh 生成秘钥
   ssh-keygen -t rsa -C "xiaoming@sina.com"
